# ARKit Remap Archive

This folder holds deprecated or one-off artifacts that are intentionally kept
for traceability but should not be treated as active release inputs.

## Contents

- `scripts/`
  Probe scripts, one-off diagnostics, and superseded tuning helpers.
- `reports/`
  Probe outputs, superseded calibration comparisons, and archived milestone
  reports that are no longer part of the active release surface.
- `data/`
  One-off diagnostic data moved out of active folders.

## Rule of thumb

If a file is no longer part of the current runtime, release package, or active
continuation workflow, prefer archiving it here rather than deleting it.
